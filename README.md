# 🚁 Simulador de Despegue de Helicóptero

Este proyecto consiste en un simulador básico de helicóptero desarrollado en **Unity** como parte del **Examen Final de la asignatura Desarrollo de Videojuegos**.

## Características

* Helicóptero construido utilizando primitivas de Unity.
* Rotación de las aspas principal y trasera.
* Incremento progresivo de la velocidad de las aspas al mantener presionada la tecla **↑ (Up Arrow)**.
* Despegue automático cuando la velocidad de las aspas supera un umbral establecido.
* Descenso controlado al soltar la tecla.

## Controles

| Tecla        | Acción                                                    |
| ------------ | --------------------------------------------------------- |
| ↑ (Up Arrow) | Aumentar la velocidad de las aspas y realizar el despegue |

## Tecnologías utilizadas

* Unity
* C#
* Unity Visual Scripting
